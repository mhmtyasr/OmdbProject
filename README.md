**Akinon  Movie Demo Project**



## Introduction

 This project is built on HTML+Jquery+Webpack

## DEMO

[Surge.sh](http://akinonproject.surge.sh/)

## Getting Started

### Installing

After cloning repository, first install npm packages:

```sh
npm install 
```

And then start

```
npm start
```
## Deployment

npm build
